### **第一题**

**【题目提取】**

> 4. 已知连续控制器 $D(s) = \frac{1}{s^2 + 0.9s + 1}$。取采样周期 $T = 0.5$ 秒。
> (1) 试分别采用前向差分法和后向差分法将其离散化；
> (2) 试分别在 S 平面和 Z 平面上画出对应极点的位置；
> (3) 说明数字控制器的稳定性。
>
> 

**【详细解析】**

**(1) 离散化**
**原理**：

* **前向差分法**利用导数的前向差商近似，即 $s \approx \frac{z-1}{T}$。
* **后向差分法**利用导数的后向差商近似，即 $s \approx \frac{z-1}{Tz}$。
* **前向差分法：**
将 $s = \frac{z-1}{0.5} = 2(z-1)$ 代入 $D(s)$：

$$D_f(z) = \frac{1}{(2(z-1))^2 + 0.9(2(z-1)) + 1}$$



展开分母：

$$D_f(z) = \frac{1}{4(z^2 - 2z + 1) + 1.8z - 1.8 + 1}$$


$$D_f(z) = \frac{1}{4z^2 - 8z + 4 + 1.8z - 0.8} = \frac{1}{4z^2 - 6.2z + 3.2}$$


* **后向差分法：**
将 $s = \frac{z-1}{0.5z} = \frac{2(z-1)}{z}$ 代入 $D(s)$：

$$D_b(z) = \frac{1}{\left(\frac{2(z-1)}{z}\right)^2 + 0.9\left(\frac{2(z-1)}{z}\right) + 1}$$



分子分母同乘 $z^2$：

$$D_b(z) = \frac{z^2}{4(z-1)^2 + 1.8z(z-1) + z^2}$$



展开分母：

$$D_b(z) = \frac{z^2}{4(z^2 - 2z + 1) + 1.8z^2 - 1.8z + z^2}$$


$$D_b(z) = \frac{z^2}{4z^2 - 8z + 4 + 1.8z^2 - 1.8z + z^2} = \frac{z^2}{6.8z^2 - 9.8z + 4}$$



**(2) 极点位置分析 (S平面与Z平面)**
**原理**：极点是使系统传递函数分母为零的根。S平面极点通过 $z = 1+Ts$（前向）或 $z = \frac{1}{1-Ts}$（后向）映射到Z平面。

* **S 平面极点**：
令 $s^2 + 0.9s + 1 = 0$。根据求根公式：

$$s_{1,2} = \frac{-0.9 \pm \sqrt{0.9^2 - 4(1)(1)}}{2} = \frac{-0.9 \pm \sqrt{-3.19}}{2} = -0.45 \pm j\frac{\sqrt{3.19}}{2} \approx -0.45 \pm j0.8938$$



*(极点位于 S 平面的左半平面，说明原连续系统稳定)*。
* **Z 平面极点（前向差分）**：
令 $4z^2 - 6.2z + 3.2 = 0 \Rightarrow z^2 - 1.55z + 0.8 = 0$。

$$z_{1,2} = \frac{1.55 \pm \sqrt{1.55^2 - 4(0.8)}}{2} = \frac{1.55 \pm \sqrt{2.4025 - 3.2}}{2} = 0.775 \pm j\frac{\sqrt{0.7975}}{2} \approx 0.775 \pm j0.4465$$



极点坐标为 $(0.775, \pm 0.4465)$。
* **Z 平面极点（后向差分）**：
令 $6.8z^2 - 9.8z + 4 = 0$。

$$z_{1,2} = \frac{9.8 \pm \sqrt{9.8^2 - 4(6.8)(4)}}{2(6.8)} = \frac{9.8 \pm \sqrt{96.04 - 108.8}}{13.6} = \frac{9.8 \pm j\sqrt{12.76}}{13.6} \approx 0.7206 \pm j0.2627$$



极点坐标为 $(0.7206, \pm 0.2627)$。

**(3) 稳定性说明**
**原理**：离散（数字）系统稳定的充要条件是闭环极点全部位于 Z 平面的单位圆内（即极点的模 $|z| < 1$）。

* **前向差分系统的稳定性**：
极点模长 $|z| = \sqrt{0.775^2 + 0.4465^2} = \sqrt{0.600625 + 0.199362} = \sqrt{0.8} \approx 0.894$。
因为 $0.894 < 1$，所有极点均在单位圆内，**该前向差分数字控制器稳定**。
* **后向差分系统的稳定性**：
极点模长 $|z| = \sqrt{0.7206^2 + 0.2627^2} = \sqrt{\frac{4}{6.8}} \approx \sqrt{0.588} \approx 0.767$。
因为 $0.767 < 1$，所有极点均在单位圆内，**该后向差分数字控制器稳定**。

### **第二题**

**【题目提取】**

> 11. 已知模拟控制器 $D(s) = \frac{10(s+1)}{s+10}$，试用频率预畸变双线性变换法将 $D(s)$ 离散化成数字控制器 $D(z)$。取 $T = 0.05$ 秒，按低通特性要求。
>
> 

**【详细解析】**
**原理**：双线性变换法虽然克服了频率混叠，但会产生频率非线性畸变。频率预畸变旨在选定一个**关键频率**（在此为低通滤波器的转折频率或截止频率），使离散前后的该频率处的频响完全一致。
预畸变双线性变换的替换公式为：

$$s = c \frac{z-1}{z+1}$$


其中畸变常数 $c = \frac{\omega_a}{\tan(\omega_a T / 2)}$。

**解题步骤**：

1. **确定预畸变频率 $\omega_a$**：
控制器 $D(s) = \frac{10(s+1)}{s+10}$ 是一个具有低通特性的超前-滞后环节，按低通特性要求，其决定带宽的关键频率（极点转折频率）为 $\omega_a = 10$ rad/s。
2. **计算畸变常数 $c$**：
取 $\omega_a = 10$，已知 $T = 0.05$ 秒。

$$c = \frac{10}{\tan\left(\frac{10 \times 0.05}{2}\right)} = \frac{10}{\tan(0.25 \, \text{rad})}$$



（注：$\tan(0.25)$ 需用弧度制计算，$\tan(0.25) \approx 0.25534$）

$$c = \frac{10}{0.25534} \approx 39.16$$


3. **代入进行离散化**：
将 $s = c \frac{z-1}{z+1}$ 代入 $D(s)$ 中：

$$D(z) = \frac{10\left( c\frac{z-1}{z+1} + 1 \right)}{c\frac{z-1}{z+1} + 10} = 10 \cdot \frac{c(z-1) + (z+1)}{c(z-1) + 10(z+1)}$$


$$D(z) = 10 \cdot \frac{(c+1)z - (c-1)}{(c+10)z - (c-10)}$$


4. **代入 $c$ 的数值**：
将 $c = 39.16$ 代入上式：

$$D(z) = 10 \cdot \frac{40.16z - 38.16}{49.16z - 29.16} = \frac{401.6z - 381.6}{49.16z - 29.16}$$



分子分母同除以 49.16 进行化简（归一化）：

$$D(z) \approx \frac{8.169z - 7.762}{z - 0.593}$$

### **第三题**

**【题目提取】**

> 13. 分别用后向差分法和阶跃响应不变法求下列模拟控制器 $G_c(s)$ 的等效数字控制器 $D(z)$。
> 14. $G_c(s) = \frac{0.2s + 1}{s + 2}$
> 15. $G_c(s) = \frac{0.8s + 1}{s(s + 3)}$
>
> 

**【详细解析】**
**原理**：

* **后向差分法**：直接用 $s = \frac{z-1}{Tz}$ 替代原传递函数中的 $s$（题目未给定 $T$，故保留参数 $T$）。
* **阶跃响应不变法（即零阶保持器等效法, ZOH）**：保证离散系统与连续系统的阶跃响应在采样点上完全相同。公式为：
$$D(z) = (1 - z^{-1}) \mathcal{Z}\left\{ \mathcal{L}^{-1}\left\{ \frac{G_c(s)}{s} \right\} \right\}$$



#### 第 1 小题：$G_c(s) = \frac{0.2s + 1}{s + 2}$

**后向差分法：**


$$D(z) = \frac{0.2\left(\frac{z-1}{Tz}\right) + 1}{\frac{z-1}{Tz} + 2}$$


分子分母同乘 $Tz$：


$$D(z) = \frac{0.2(z-1) + Tz}{z-1 + 2Tz} = \frac{(0.2+T)z - 0.2}{(1+2T)z - 1}$$

**阶跃响应不变法：**

1. 构造带有阶跃输入的传递函数并进行部分分式展开：

$$\frac{G_c(s)}{s} = \frac{0.2s + 1}{s(s + 2)} = \frac{A}{s} + \frac{B}{s + 2}$$



利用留数法求解：
$A = \left. \frac{0.2s+1}{s+2} \right|_{s=0} = \frac{1}{2} = 0.5$
$B = \left. \frac{0.2s+1}{s} \right|_{s=-2} = \frac{-0.4+1}{-2} = -0.3$
所以，$\frac{G_c(s)}{s} = \frac{0.5}{s} - \frac{0.3}{s + 2}$
2. 求对应的 Z 变换：

$$\mathcal{Z}\left\{ \mathcal{L}^{-1}\left\{ \frac{G_c(s)}{s} \right\} \right\} = \frac{0.5z}{z-1} - \frac{0.3z}{z - e^{-2T}}$$


3. 乘以 $(1 - z^{-1}) = \frac{z-1}{z}$：

$$D(z) = \frac{z-1}{z} \left( \frac{0.5z}{z-1} - \frac{0.3z}{z - e^{-2T}} \right) = 0.5 - \frac{0.3(z-1)}{z - e^{-2T}}$$



通分化简：

$$D(z) = \frac{0.5(z - e^{-2T}) - 0.3(z-1)}{z - e^{-2T}} = \frac{0.2z + 0.3 - 0.5e^{-2T}}{z - e^{-2T}}$$



#### 第 2 小题：$G_c(s) = \frac{0.8s + 1}{s(s + 3)}$

**后向差分法：**


$$D(z) = \frac{0.8\left(\frac{z-1}{Tz}\right) + 1}{\frac{z-1}{Tz} \left( \frac{z-1}{Tz} + 3 \right)}$$


分子整理：$ \frac{0.8(z-1) + Tz}{Tz} $
分母整理：$ \frac{z-1}{Tz} \cdot \frac{z-1 + 3Tz}{Tz} = \frac{(z-1)((1+3T)z - 1)}{T^2z^2} $
两者相除：


$$D(z) = \frac{\frac{(0.8+T)z - 0.8}{Tz}}{\frac{(z-1)((1+3T)z - 1)}{T^2z^2}} = \frac{Tz \cdot ((0.8+T)z - 0.8)}{(z-1)((1+3T)z - 1)}$$


展开分母：


$$D(z) = \frac{(0.8T + T^2)z^2 - 0.8Tz}{(1+3T)z^2 - (2+3T)z + 1}$$

**阶跃响应不变法：**

1. 构造传递函数并展开：

$$\frac{G_c(s)}{s} = \frac{0.8s + 1}{s^2(s + 3)} = \frac{A}{s^2} + \frac{B}{s} + \frac{C}{s+3}$$



$A = \left. \frac{0.8s+1}{s+3} \right|_{s=0} = \frac{1}{3}$
$C = \left. \frac{0.8s+1}{s^2} \right|_{s=-3} = \frac{-2.4+1}{9} = -\frac{1.4}{9} = -\frac{7}{45}$
$B = \left. \frac{d}{ds}\left(\frac{0.8s+1}{s+3}\right) \right|_{s=0} = \left. \frac{0.8(s+3) - (0.8s+1)}{(s+3)^2} \right|_{s=0} = \frac{2.4 - 1}{9} = \frac{1.4}{9} = \frac{7}{45}$
所以，$\frac{G_c(s)}{s} = \frac{1/3}{s^2} + \frac{7/45}{s} - \frac{7/45}{s+3}$
2. 求 Z 变换：

$$\mathcal{Z}\{\dots\} = \frac{1}{3} \frac{Tz}{(z-1)^2} + \frac{7}{45} \frac{z}{z-1} - \frac{7}{45} \frac{z}{z - e^{-3T}}$$


3. 乘以 $\frac{z-1}{z}$：

$$D(z) = \frac{1}{3} \frac{T}{z-1} + \frac{7}{45} - \frac{7}{45} \frac{z-1}{z - e^{-3T}}$$



提取公因式并通分：

$$D(z) = \frac{15T(z - e^{-3T}) + 7(z-1)(z - e^{-3T}) - 7(z-1)^2}{45(z-1)(z - e^{-3T})}$$



分子展开化简：
$= 15Tz - 15Te^{-3T} + 7(z^2 - (1+e^{-3T})z + e^{-3T}) - 7(z^2 - 2z + 1)$
$= (15T - 7 - 7e^{-3T} + 14)z - 15Te^{-3T} + 7e^{-3T} - 7$

$$D(z) = \frac{(15T + 7 - 7e^{-3T})z - (7 - 7e^{-3T} + 15Te^{-3T})}{45(z-1)(z - e^{-3T})}$$

### **第四题**

**【题目提取】**

> 二、(12分) 已知单位负反馈计算机控制系统的开环 Z 传递函数为：
>
> $$G(z) = \frac{2z^2 - 3z + 1}{z^3 - 4z^2 + 7z - 1}$$
>
> 
>
> 试判别其闭环系统的稳定性。

**【详细解析】**
**原理**：判定闭环稳定性需要求出闭环特征方程，并检查其所有的根（极点）是否都严格位于 Z 平面的单位圆内（$|z| < 1$）。

**解题步骤**：

1. **列出闭环特征方程**：
对于单位负反馈系统，闭环特征方程为 $1 + G(z) = 0$。

$$1 + \frac{2z^2 - 3z + 1}{z^3 - 4z^2 + 7z - 1} = 0$$



化简得到：

$$z^3 - 4z^2 + 7z - 1 + 2z^2 - 3z + 1 = 0$$


$$z^3 - 2z^2 + 4z = 0$$


2. **求解特征根（闭环极点）**：
提取公因式 $z$：

$$z(z^2 - 2z + 4) = 0$$



由此得到三个根：
* $z_1 = 0$
* 对于 $z^2 - 2z + 4 = 0$，运用求根公式：

$$z_{2,3} = \frac{2 \pm \sqrt{4 - 16}}{2} = \frac{2 \pm \sqrt{-12}}{2} = 1 \pm j\sqrt{3}$$




3. **判断稳定性**：
计算各极点的模长：
* $|z_1| = 0 < 1$
* $|z_{2,3}| = \sqrt{1^2 + (\sqrt{3})^2} = \sqrt{1 + 3} = \sqrt{4} = 2$
因为极点 $z_{2,3}$ 的模长大于 1（处于单位圆之外），所以**该闭环系统是不稳定的**。

### **第五题**

**【题目提取】**

> 五、(10分) 某系统的闭环特征多项式为 $F(z) = z^3 + 1.8z^2 + 1.01z + 0.18$，请判定该系统的稳定性，并说明该系统的特征根相对于半径为 $0.8$ 且和单位圆同心的圆的分布。

**【详细解析】**
**原理**：
第一步判定系统稳定性（分析特征根是否在单位圆 $|z|=1$ 内部）。可以通过求根或 Jury 判据完成。第二步分析分布，则是比较各特征根模长与 $0.8$ 的关系。这道题系数非常特殊，可以直接通过因式分解求出精确的闭环极点，这将使分析极为直观。

**解题步骤**：

1. **直接求解特征根**：
观察多项式 $F(z) = z^3 + 1.8z^2 + 1.01z + 0.18 = 0$。
我们可以尝试代入一些常见值或有理根。注意到多项式各项系数全为正，故实数根必为负数。
尝试代入 $z = -0.9$：

$$F(-0.9) = (-0.9)^3 + 1.8(-0.9)^2 + 1.01(-0.9) + 0.18$$


$$= -0.729 + 1.8(0.81) - 0.909 + 0.18$$


$$= -0.729 + 1.458 - 0.909 + 0.18$$


$$= 1.638 - 1.638 = 0$$



既然 $z = -0.9$ 是一个根，则 $(z + 0.9)$ 是多项式的一个因式。利用多项式长除法降幂：

$$\frac{z^3 + 1.8z^2 + 1.01z + 0.18}{z + 0.9} = z^2 + 0.9z + 0.2$$



因此，特征方程可分解为：

$$(z + 0.9)(z^2 + 0.9z + 0.2) = 0$$



解二次方程 $z^2 + 0.9z + 0.2 = 0$：

$$z = \frac{-0.9 \pm \sqrt{0.81 - 0.8}}{2} = \frac{-0.9 \pm 0.1}{2}$$



得到另外两个根为 $z = -0.4$ 和 $z = -0.5$。
综上，**系统的三个特征根分别为：$z_1 = -0.9$、$z_2 = -0.5$、$z_3 = -0.4$**。
2. **判定系统的稳定性**：
由于 $|z_1| = 0.9 < 1$，$|z_2| = 0.5 < 1$，$|z_3| = 0.4 < 1$。
所有特征根均位于 Z 平面的单位圆内部。
因此，**该闭环系统是稳定的**。
3. **说明特征根相对于半径为 0.8 圆的分布**：
我们将各个根的模长与 $0.8$ 比较：
* $|z_1| = |-0.9| = 0.9 > 0.8$ （位于半径为 $0.8$ 的圆**外**）
* $|z_2| = |-0.5| = 0.5 < 0.8$ （位于半径为 $0.8$ 的圆**内**）
* $|z_3| = |-0.4| = 0.4 < 0.8$ （位于半径为 $0.8$ 的圆**内**）

**结论**：在系统的三个特征根中，有 **1 个**分布在半径为 0.8 的圆周与单位圆之间（即 0.8 圆外，单位圆内）；有 **2 个**分布在半径为 0.8 的圆周内部。
